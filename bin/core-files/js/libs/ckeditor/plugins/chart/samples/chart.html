CKEditor Samples » Chart Plugin

This editor is using the Chart plugin which introduces charts into CKEditor. The chart plugin depends on the built-in client-side library Chart.js.

CKEditor returns a div element, which needs to be converted into a chart by Chart.js. This is done automatically, after adding two JavaScript files to a website.
